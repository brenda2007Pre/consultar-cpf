Claro! Abaixo está um exemplo simples de um site HTML + JavaScript que simula a página de consulta de CPF parecida com o site do gov.br, mas de forma fictícia e sem conexão real com dados do governo.

✅ Importante:

Essa simulação não acessa dados reais nem deve ser usada para fins oficiais. É apenas para aprendizado ou demonstração.

✅ Resultado esperado:

  • Se o CPF for 122.198.106-40, ele mostrará: "Data de nascimento: 24/01/2007".

  • Para qualquer outro CPF, ele mostrará: "CPF não encontrado."

💻 Código do site (HTML + CSS + JS):

html
<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Consulta de CPF - Simulação</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      background-color: #f3f3f3;
      margin: 0;
      padding: 0;
    }

    .container {
      max-width: 500px;
      margin: 80px auto;
      background-color: white;
      padding: 30px;
      border-radius: 8px;
      box-shadow: 0 0 10px rgba(0,0,0,0.1);
    }

    h1 {
      font-size: 22px;
      color: #005aa7;
      margin-bottom: 20px;
    }

    label {
      display: block;
      margin-bottom: 10px;
      font-weight: bold;
    }

    input[type="text"] {
      width: 100%;
      padding: 10px;
      margin-bottom: 20px;
      border: 1px solid #ccc;
      border-radius: 4px;
    }

    button {
      padding: 10px 20px;
      background-color: #005aa7;
      color: white;
      border: none;
      border-radius: 4px;
      cursor: pointer;
    }

    button:hover {
      background-color: #004080;
    }

    .resultado {
      margin-top: 20px;
      font-weight: bold;
    }
  </style>
</head>
<body>

  <div class="container">
    <h1>Consultar CPF (Simulado)</h1>
    <form id="cpfForm">
      <label for="cpf">Digite o CPF:</label>
      <input type="text" id="cpf" placeholder="Ex: 122.198.106-40" required>
      <button type="submit">Consultar</button>
    </form>
    <div class="resultado" id="resultado"></div>
  </div>

  <script>
    document.getElementById('cpfForm').addEventListener('submit', function(e) {
      e.preventDefault();

      const cpfInput = document.getElementById('cpf').value.trim();
      const resultado = document.getElementById('resultado');

      if (cpfInput === '122.198.106-40') {
        resultado.textContent = 'Data de nascimento: 24/01/2007';
      } else {
        resultado.textContent = 'CPF não encontrado.';
      }
    });
  </script>

</body>
</html>
